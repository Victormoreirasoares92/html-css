Outros tipos de formatação.

Texto Marcado.

Podemos usar um texto marcado usando a tag mark ex: nessa frase

Para mudar a cor da marcação voce usa a tag MARK e usa a tag Style = backgroun-color depois da tag Style para mudar a cor do marcador . ex: esse esta verde

Texto grande e pequeno.

Estamos criando um texto grande e um texto pequeno nessa frase.

Texto deletado.

Podemos marcar um texto como Excluido para indicar para que ele deve ser lido e nao considerado (usando a tag del)

Texto Inserido.

Podemos marcar um texto como Incluido para indicar que ele foi adicionado depois.(usando a tag ins)

Existe tambem o sublinhado com a tag u ( que é não semantica)

Texto Sobreescrito.

Para inserir coisas do tipo x20+3 (usando a tag sup)

Texto Subecrito.

Para inserir coisas como H2O (usando a tag sub)
