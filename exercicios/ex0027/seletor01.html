Utilizando a tag id

Modificando apenas uma linha

Voce pode alterar a pagina inteira do seu site e modificar apenas uma linha sem ter que ir nela ou criar o seletor para usar a mesma modificação, seja de fonte ou de tamanho ou cor, usando a tag id e colocando o nome do seletor, essa configuração para voce persolalizar seu texte sem ter que ir linha por linha e modifica-lo e assim poder fazer as modificações link do css criado.

Tudo que em HTML é id = em CSS é #

Então voce cria a ID e depois põe o nome dela depois da #

Usando a tag CLASS

Modificando usando class

Voce pode usar a tag class para poder criar uma classe moficadora ou seja para voce alterar mais de uma tag sem ter que selecionar uma por uma e digitar a configuração. Voce pode criar mais de uma class no mesmo documento e colocar id e class no mesmo elemento

Exemplo : class="teste teste1" ou id = "principal" class "teste1"

Tudo que em HTML é class = em CSS é .
